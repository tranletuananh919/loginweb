Demo: Login với Google

Chạy được trên GitHub Pages vì không cần backend. Chỉ là demo hiển thị thông tin cơ bản từ ID Token.

Sau khi đăng nhập, trang sẽ giải mã ID token trên trình duyệt để hiện tên, email, ảnh đại diện. Đây chỉ là demo — nếu muốn bảo mật thật sự, bạn cần có backend để xác thực ID token với Google.
